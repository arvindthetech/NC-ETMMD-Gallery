This is the official **Photo Gallery** for the **National Conference on Emerging Trends in Multifunctional Materials and Devices (NC-ETMMD-2026)** organized by the **School of Physical Sciences, Jawaharlal Nehru University, New Delhi**.

The gallery features **316+ high-quality photos** capturing memorable moments from the conference held on **March 11-12, 2026**.
